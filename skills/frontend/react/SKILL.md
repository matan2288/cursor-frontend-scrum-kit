# Frontend React

Use this skill when implementing a new frontend feature, component, UI behavior, or focused bug fix in a React + TypeScript project.

## Mission

Deliver working code that fits the local codebase. Prefer the project's existing architecture, styling strategy, test approach, and naming conventions over generic best practices.

## First Step

Before editing, quickly inspect local patterns: similar components, prop/state/hook conventions, the styling approach, existing tests/stories, and the scripts in `package.json`. If requirements are ambiguous, ask the smallest clarifying question; otherwise use a sensible codebase default and state the assumption.
